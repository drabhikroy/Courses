Submission

eCampus

Type

Team Submission. Students in the course must participate in the recording process in a group oriented setting.

Begin Work

to fill in some parts as early as possible

at the latest after the conclusion of the evaluation

Criteria

Create and disseminate an evaluation presentation1 using the criteria outlined below. This submission is due in one week prior to the final week of the term. For an exact date, please check eCampus.

Structure

  1. construct a single presentation that uses a one voice

  2. use language appropriate for the class audience and using proper grammar, spelling, and consistency2 in tone

  3. make the product aesthetically pleasing

  4. remain consistent in how elements of the evaluation are presented and discussed

  5. Use a format of your choice

  6. name your file teamname_presentation.typeoffile


Body

The presentation3 must be delivered using an audio/video medium constructed by a software of your choice with equal contributions from each member and is publicly accessible

  1. descriptions of the program

  2. a narrative on the entire evaluative process

  3. data visualizations

  4. instructional materials

  5. specific prompts for the rest of the class to consider.


It is highly unlikely that a score will be changed unless there is an egregious error on my end so please check your entire presentation prior to submission.
